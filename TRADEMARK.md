# Trademark Guidelines

The Apache 2.0 license covers the source code in this repository. It does not
grant trademark rights in the latch name, logos, or brand identifiers.

You may:

- say your work is based on latch,
- say your project is compatible with latch,
- use the latch name in factual references to this repository.

You may not, without written permission:

- present a fork as the official latch project,
- use latch branding in a way that suggests endorsement or sponsorship,
- register confusingly similar project, package, domain, or product names,
- use latch marks for a service or distribution in a way that implies it is the
  official project.

Forks and downstream packages should use their own names and clearly state that
they are independent from the official latch project.
